# Language

- **MUST** 預設情況下，所有回應皆使用**繁體中文**，包含但不限於：對話內容、文件撰寫、程式註解與備註等
- **SHOULD** 專業術語若存在通用的**繁體中文譯名**，優先使用繁體中文；若無慣用或明確譯名，則保留**原文術語**

## Standards

**MUST** 語言完整規範 [zh-tw.md](.standards/zh-tw.md)
